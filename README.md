# ERPL Website
 Website Design/Creation for Experimental Rocket Prop Lab

## Styling
 - Shall follow [ERPL's Font/Color/Logo standards](https://docs.google.com/document/d/10Wh_7K3pstGV6kjZu7W7w3cRbqlsxaehFLRp65ad1p4/edit "Website Format requirements")
 - Shall be [responsive](https://www.w3schools.com/css/css_rwd_intro.asp "Responsive Design") 
     - Mobile first, Tablet/Low Desktop second, High-res Desktop third
 - Shall have an always visible nav-bar
 - Shall have a footer at the bottom of all pages
 - Should be SEO optimized using Google's Lighthouse
 - Should prioritize chromium over safari & explorer
